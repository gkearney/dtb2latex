DTB2LaTex

DTB2LaTex is a small utility program for conveting DTBook xml files to LaTeX files for the production of large pint books in PDF format.

Program usage

When first run DTB2LaTex will display a window with three controls on it.

Font
This is a popup menu of font choices
Size
This is a popup menu of font size choices. Note that for large type the default of 17 points should be used.
Paper
This is a popup menu of paper size choices.

Once you have made your selection from these options you then use the Convert... menu option of the File menu and select your DTbook XML file. You will then be ask where to save the LaTeX file. Conversion will take place at once and you will be see a conversion window while it is taking place. The resuting LaTeX file will be opened when the conversion is complete.
